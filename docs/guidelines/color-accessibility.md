# Guia de acessibilidade visual — Paletas e contraste

Diretrizes para escolher cores legíveis e seguras para daltonismo.

1. Use paletas com contraste adequado (relação de contraste >= 4.5:1 para texto pequeno)
2. Prefira paletas com distinção por forma e textura além da cor (linhas tracejadas, marcadores distintos)
3. Teste para protanopia, deuteranopia e tritanopia usando ferramentas como ColorBrewer e simuladores
4. Evite usar exclusivamente vermelho/verde para diferenciação de categorias
5. Se necessário, adicione anotações diretas (ex.: nomes dentro das áreas) para evitar ambiguidades
