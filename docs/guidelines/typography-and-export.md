# Tipografia e padrões de exportação

Recomendações rápidas para tipografia e entrega de arquivos.

- Tipografia: fontes sem serifa para labels (ex.: Inter, Helvetica), contraste claro entre texto e fundo
- Tamanhos: títulos (10–12 pt), labels (8–10 pt), ticks (6–8 pt) dependendo do tamanho final da figura
- Espaçamento: margens internas consistentes entre painéis (5–10 px em figuras digitais, ajustar para print)
- Exportação:
  - PNG: 300–450 dpi para print
  - SVG: vetorial para web e edição
  - PDF/TIFF: export para submissão em alta qualidade
  - Fornecer arquivo editável (AI, EPS ou fonte embutida no SVG)

Padrão de entrega: pacote com <figure-name>_web.png, <figure-name>_print.tiff, <figure-name>.svg, <figure-name>_source.zip
